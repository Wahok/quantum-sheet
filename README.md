## For Developers

1. Clone the project
2. `npm install`
3. `npm run download:pyodide`
4. `npm run dev`

Check out [CONTRIBUTING.md](https://github.com/stefnotch/quantum-sheet/blob/master/CONTRIBUTING.md) for more
